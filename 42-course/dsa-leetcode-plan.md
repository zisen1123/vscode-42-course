# 数据结构 + LeetCode 并行训练计划

LeetCode 仓库：`https://github.com/zisen1123/leetcode-master.git`

目标：把数据结构训练并入 42-course 主线，避免只学语法不做算法。

## 每周时间分配（总 9-11h）

- 主线课程（C/ML）：6-7h
- 数据结构 + LeetCode：3-4h

## 执行规则

- 每周固定 6 题：
  - 3 题 `easy`
  - 2 题 `medium`
  - 1 题复盘题（重做旧错题）
- 每题必须记录：
  - 思路
  - 时间复杂度和空间复杂度
  - 易错点
- 每周至少 1 次“限时 45 分钟模拟”

## 24 周刷题主题（与 42-course 同步）

1. 数组与双指针
2. 字符串与哈希
3. 栈与队列
4. 链表基础
5. 链表进阶（反转、环、相交）
6. 二叉树遍历
7. 二叉树递归题
8. BST 题型
9. 回溯基础
10. 回溯剪枝
11. 二分搜索
12. 滑动窗口
13. 堆与优先队列
14. 贪心基础
15. 动态规划入门
16. 线性 DP
17. 背包 DP
18. 区间 DP / 状态机 DP
19. 图遍历（DFS/BFS）
20. 拓扑排序与并查集
21. 最短路径
22. 单调栈与单调队列
23. 高频面试混合题
24. 错题清算周（只刷错题）

## 与当前周计划的联动方式

- `week1-week6`：重点刷数组、字符串、链表，代码用 C 实现优先。
- `week7-week12`：继续刷树、回溯、二分，保持每周 6 题。
- `week13-week20`：配合 C 深化，刷哈希、栈队列、图。
- `week21-week36`：配合 ML 深化，刷 DP、图、综合题，强化面试能力。
